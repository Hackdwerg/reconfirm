# reconfirm
Reconfirm - a small tool that helps you during your recon process

## Usage:

./reconfirm -t <IP>
  
./reconfirm -g (some tips and tricks) 

./reconfirm -h (Help menu)

./reconfirm -v (version of software)

## example
./reconfirm -t 10.11.1.1
